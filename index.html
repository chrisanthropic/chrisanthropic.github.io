Tools

  • Ansible AWS Template - A tiny Ansible quick-start for AWS that builds: 1 VPC w/an Internet Gateway, 2 subnets, 2 t2.micro instances, and 2 EC2 security groups.
  • Open Publisher - Write your manuscript in Markdown and export as beautifully designed epub, mobi, and print-ready 5x8 & 6x9 pdf.
  • debian2kali - Packer template to create your own Kali Linux AWS HVM AMI. Kali's AWS AMI offerings have been hit-and-miss. This template reverse engineers the Kali build process, starting with a standard debian base, Kali's 'preseed' steps, and Kali's repos.

Docker Containers

  • docker-alpine-rpi-jekyll - An Alpine-based container to run Jekyll on a Raspberry Pi.
  • docker-alpine-rpi-jenkins - An Alpine-based container to run Jenkins on a Raspberry Pi.
  • docker-alpine-rpi-glibc - An Alpine-based container with Glibc installed for Raspberry Pi.
  • docker-kali - A full Kali desktop in a container.
  • docker-kali-msf - A single container that when launched will start a Kali linux machine with postgresql and the metasploit framework and will drop you into the msfconsole.
  • docker-Open-Publisher-Base - A container to turn markdown into ready-to-publish manuscripts.
